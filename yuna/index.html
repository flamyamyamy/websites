Yuna Bot

$ Advanced Discord Moderation & Utility Bot

Yuna is a modern Discord bot built for communities of all sizes, offering advanced moderation systems, leveling mechanics, anime integrations, social interaction features, and scalable utility tools — all designed with performance, usability, and reliability in mind.

Support Server View Commands

// Command Directory

⛩️ Anime & Manga

/anime <title>
Shows anime information including synopsis, status and episodes.

/character <name>
Displays anime character information and artwork.

/manga <title>
Shows manga data and publication details.

👤 User Utilities

/userinfo [user]
Shows Discord profile information.

/avatar [user]
Displays user avatar in full resolution.

/banner [user]
Displays user banner.

📊 Level System

/rank
Shows rank card and XP progress.

/leaderboard
Displays the most active users.

/level-setup
Configure the XP system (Admin only).

📜 Quotes

/quote
Returns a random inspirational quote.

🎮 Fun Commands

/ship <user1> <user2>
Calculates compatibility between two users.

/react <user>
Sends anime reaction GIFs.

💖 Reactions

Use /react with supported reaction types to interact using animated anime GIFs and expressive community actions.
